# ResearchWise
A simple research agent using a supervisor-style agent pattern capable of web research and fact-checking.

![Architecture](image.png)

Libraries used:
1. [Tavily](https://docs.tavily.com/documentation/quickstart)
2. [Langchain](https://docs.langchain.com/oss/python/langgraph/functional-api#detailed-explanation)
3. [Fast API](https://fastapi.tiangolo.com)
